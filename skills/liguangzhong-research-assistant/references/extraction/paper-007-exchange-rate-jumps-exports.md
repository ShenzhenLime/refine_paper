## B. 一句话摘要

为了检验汇率跳跃风险（而非连续波动）对出口贸易的影响，作者使用 ARJI 模型提取 81 国 1995–2004 年双边汇率跳跃强度（利用人民币钉住美元期间的外生性），以重力方程和 PPML 估计发现：汇率跳跃风险对出口有显著的额外负向影响，且国内金融发展无法缓解这一负效应。

## C. 研究问题

| 维度 | 内容 |

| 表层问题 | 汇率跳跃风险是否影响出口贸易？ |
| 深层问题 | 传统汇率波动率只捕捉了连续波动，跳跃风险作为不可对冲的风险是否对贸易有独立影响？金融发展能否缓解？ |
| X | 双边汇率跳跃强度（ARJI 模型估计） |

| M | 国内金融发展（与跳跃强度的交互项） |
| 分析单位 | 国家-年份（双边贸易） |
| 问题类型 | 因果效应（利用钉住汇率的外生性） |

## D. 识别亮点

1. 1995–2004 年人民币钉住美元，双边汇率波动对其他货币完全外生于中国经济
2. ARJI 模型分离连续波动和跳跃
3. PPML 处理零贸易流和异方差
4. 三重修正：零值、异方差、内生性

## E. 核心发现

- 汇率跳跃强度对出口有显著负向影响（超出传统波动率度量的额外解释力）
- 国内金融发展促进贸易，但不能缓解跳跃风险的负面影响
