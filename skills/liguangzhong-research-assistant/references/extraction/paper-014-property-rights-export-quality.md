# Paper 014: Does property rights protection affect export quality? Evidence from a property law enactment

## B. 一句话摘要

为了检验产权保护对企业出口产品质量的影响，作者以 2007 年中国《物权法》的颁布实施为自然实验，利用中国海关交易级数据和企业数据构建 DID 模型，发现产权保护的加强显著提高了企业出口产品质量，且该效应在融资约束严重、无形资产占比高的企业中更显著。

## C. 研究问题

| 维度 | 内容 |
|---|---|
| 表层问题 | 产权保护是否影响出口产品质量？ |
| 深层问题 | 法律制度（产权保护）如何通过缓解融资约束和激励无形资产投资来影响企业的出口质量升级？ |
| X | 《物权法》颁布（外生法律冲击） |
| Y | 出口产品质量（单价、质量指数） |
| M | 融资约束缓解、无形资产投资激励 |
| 分析单位 | 企业-产品-年份（交易级） |
| 问题类型 | 因果效应 + 机制解释 |
| 为什么重要 | 连接"法与金融"和"国际贸易"两个文献 |
| 为什么困难 | 需要排除同期其他政策干扰；出口质量度量 |

## D. 识别策略

| 要素 | 内容 |
|---|---|
| 自然实验 | 2007 年《物权法》：首次系统保护私有财产 |
| 外生性 | 法律由全国人大颁布，企业无法操控 |
| 处理组 vs 对照组 | 高无形资产/高融资约束企业 → 更受物权法影响 |
| 方法 | DID + 连续处理强度 |
| 数据 | 中国海关交易级数据 + 工业企业数据库 |
| 被解释变量 | 出口产品质量（多种度量） |
| 稳健性 | 平行趋势检验、安慰剂、替换度量 |

## E. 核心发现
1. 《物权法》显著提高出口产品质量
2. 机制：缓解融资约束（企业可获得更多外部融资→投资质量升级）+ 保护无形资产回报
3. 异质性：高无形资产行业、高融资约束企业效应更强
