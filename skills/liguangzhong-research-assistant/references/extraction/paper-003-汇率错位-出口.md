# Paper 003: 实际汇率错位、汇率波动性及其对制造业出口贸易影响

## A. 元信息

| 项目 | 内容 |
|---|---|
| 标题 | 实际汇率错位、汇率波动性及其对制造业出口贸易影响的实证分析：1978–1998年平行数据研究 |
| 作者 | 李广众（独作） |
| 年份 | ~2003 |
| 期刊 | 管理世界 |
| 研究领域 | 汇率、国际贸易 |
| 方法类型 | SUR（似不相关回归）/ 面板 |
| 识别强度 | 弱-中 |
| 与 Skill 相关度 | 核心（汇率主题奠基之作） |

## B. 一句话摘要
利用 SITC 3 位数代码商品 1978–1998 年对不同国家出口的平行数据，采用 SUR 方法估计出口需求方程系统，同时考察实际汇率水平、汇率波动性和汇率错位三个维度对出口的影响，并模拟了汇率制度变动的净影响。

## C. 核心创新
- 同时对三个汇率维度建模（水平、波动性、错位），避免遗漏变量偏误
- 提出"汇率错位"应纳入出口方程（前人研究仅关注波动性）
- SUR 系统估计处理跨方程相关性
- 模拟分析：汇率制度改革对不同商品出口的差异化影响

## D. 可学习点
1. 汇率政策的多维度分析框架
2. SUR 方法在贸易方程中的应用
3. 从政策模拟角度给出启示
