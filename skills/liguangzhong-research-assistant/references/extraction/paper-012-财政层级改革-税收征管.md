## B. 一句话摘要

为了检验财政层级改革带来的市级政府财政压力能否激励其加强税收征管，作者以财政"省直管县"改革为外生冲击，基于 1998–2007 年中国工业企业数据，构建多时点 DID 模型发现：改革后市辖企业避税程度显著降低，且该效应仅在市级政府拥有征管权限的企业中存在。

## C. 研究问题

| 维度 | 内容 |
|---|---|
| 表层问题 | 财政"省直管县"改革是否影响企业避税行为？ |
| 深层问题 | 外生财政压力能否重构地方政府的税收征管激励？税收征管与企业避税是否存在因果关系？ |
| X | 财政"省直管县"改革（外生冲击） |
| Y | 企业避税程度 |

| 分析单位 | 企业-年份 |
| 问题类型 | 因果效应 + 机制解释 |
| 为什么重要 | 直接关系国家治理能力和财政可持续性 |
| 为什么困难 | 政府税收征管动机难以直接观测 |

## D. Introduction 结构

| 段落/功能 | 原文作用 |
|---|---|
| 开场 | 税收汲取能力是国家治理能力的基础，企业逃税严重削弱国家能力 |
| 理论框架 | 政府作为"最大的中小股东"有动机征税 |
| 识别策略 | 省直管县改革→市级财政压力→外生激励变化→准自然实验 |
| 贡献 | 首次探讨财政层级改革如何重构政府征管激励，以自然实验提供因果证据 |

## E. 识别策略

| 要素 | 内容 |
|---|---|
| 改革内容 | 省-县直接对接财政（跳过地级市），市级政府丧失县级税收分成 |
| 外生性 | 改革由省政府决策，企业无法影响 |
| 处理组 | 经历改革的市辖企业 |
| 对照组 | 未经历改革的市辖企业 |
| 方法 | 多时点 DID（不同省份在不同年份推开改革） |
| 被解释变量 | 企业避税（账面-实际税率差异、会计-应税收入差异） |
| 稳健性 | 动态效应（事件研究法）、安慰剂检验、PSM-DID、子样本 |

## F. 核心发现

1. 省直管县改革后，市辖企业避税显著减少
2. 仅对市级政府有征管权限的企业有效
3. 市级土地财政能力弱时，效应更强

## G. 可学习性评估

### 可直接学习
1. 多时点 DID + 事件研究法的完整范式
2. 利用中国制度特征（财政层级改革）构建自然实验
3. 以"权限范围"做安慰剂/证伪检验
4. 政府行为→企业行为的跨层级因果识别
