## B. 一句话摘要

为了回答"不确定性如何影响投资"这一经典问题，作者使用 1998–2006 年中国制造业企业大面板数据，以行业层面汇率波动率为不确定性度量，运用 System GMM 发现：汇率不确定性对高劳动份额、低投资不可逆性、低市场势力的企业促进了投资（即验证了 Hartman–Abel 效应），从而强调了劳动份额、投资不可逆性和市场势力三者在不确定性-投资关系中的交互作用。

## F. 理论与机制

| 机制链条 | 具体内容 |
|---|---|
| 行为主体 | 制造业企业（利润最大化） |
| 约束条件 | 投资调整成本（不可逆性）、产品市场竞争 |
| 传导渠道 | 渠道1（凸性效应）：高劳动份额 → 利润函数更凸 → 不确定性增加边际资本价值 → 促进投资 |
| | 渠道2（期权价值效应）：高不可逆性 → 等待期权价值高 → 不确定性抑制投资 |
| | 渠道3（市场势力）：高市场势力 → 减弱凸性效应 + 增强期权效应 → 抑制投资 |
| 理论预测 | 高劳动份额 + 低不可逆性 + 低市场势力 → 正向关系最强 |
| 可检验性 | 高（分组建模 + 交互项） |

## I. 实证模型与识别

### 基准模型
```
I_{i,j,t}/K_{i,j,t-1} = β₁ σ_Δier_{j,t-1} + β₂ Δier_{j,t-1} + β₃ I_{i,j,t-1}/K_{i,j,t-2} 
                      + β₄ Sale_{i,j,t-1}/K_{i,j,t-1} + β₅ CF_{i,j,t}/K_{i,j,t-1} + μ_i + η_t + ε
```

### 识别评估
| 问题 | 判断 |
|---|---|
| 核心识别假设 | 行业层面汇率不确定性对企业个体投资外生 |
| 最大内生性风险 | 汇率波动可能与企业绩效相关（通过宏观渠道） |
| 缓解策略 | 行业层面变量（非企业层面）；固定贸易权重（1998年）；System GMM处理动态面板 |
| 该策略是否足够？ | 中（未使用自然实验/DID，但行业-企业层级分离提供了可信的外生变异） |
| 因果表述边界 | "evidence suggests"级别，非纯净因果识别 |

## J. 结果与表格

| 表格 | 功能 | 核心结果 |
|---|---|---|
| 基准回归 | 全样本σ→I | 不确定性的平均效应弱/不显著 |
| 分组：劳动份额 | 高劳动份额 vs 低劳动份额 | 高组显著为正，低组显著为负 |
| 分组：不可逆性 | 高不可逆性 vs 低不可逆性 | 低组显著为正 |
| 分组：市场势力 | 高markup vs 低markup | 低组显著为正 |
| 三重交互 | 高劳动+低不可逆+低势力 | 正向效应最强 |
| 稳健性 | 替换变量、子样本 | 结论稳健 |
