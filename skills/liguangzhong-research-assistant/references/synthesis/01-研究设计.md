# 研究设计 — 李广众论文共性提炼

> 基于 15 篇入选论文提炼。只包含 ≥2 篇论文出现的共性模式。

## 0. 研究领域概况

| 领域 | 稳定性 | 代表论文 |
|---|---|---|
| 汇率与国际金融 | 稳定范式 (≥5篇) | #3, #6, #7, #8, #10 |
| 公司金融与治理 | 稳定范式 (≥4篇) | #4, #5, #9, #13 |
| 政府治理与税收 | 倾向性 (2篇) | #11, #12 |
| 法与金融/产权 | 倾向性 (1篇核心) | #14 |
| 资本市场与金融机构 | 倾向性 (≥3篇) | #4, #5, #15 |

交叉趋势明显：汇率×企业行为（#7, #10）、政府×公司治理（#12, #13）、法律×贸易（#14）。

## 1. 问题意识

### 典型问题公式
> 在 **[制度/政策/市场]** 场景下，**[外生冲击/核心变量 X]** 如何通过 **[机制 M]** 影响 **[主体]** 的 **[结果 Y]**？

### 常见 X 类型

| X 类型 | 稳定性 | 示例 |
|---|---|---|
| 汇率/汇率风险（波动性、跳跃、错位） | 稳定范式 | #3, #6, #7, #8, #10 |
| 政策冲击（法律、改革、监管） | 稳定范式 | #12, #13, #14, #15 |
| 制度特征（劳动保护、产权） | 倾向性 | #11, #14 |
| 公司治理变量（激励、转移） | 倾向性 | #4, #9 |

### 常见 Y 类型

| Y 类型 | 稳定性 |
|---|---|
| 企业投资/生产率 | 稳定范式（#1, #10, #15） |
| 企业出口/贸易量 | 稳定范式（#3, #7, #14） |
| 企业避税/盈余管理 | 倾向性（#12, #13） |
| 企业利润/价值 | 倾向性（#8, #9） |
| 工资/劳动收入 | 倾向性（#11, #15 劳动收入份额） |

### 常见 M 链条

| 机制链条 | 稳定性 | 论文 |
|---|---|---|
| 汇率不确定性 → 凸性效应/期权效应 → 企业投资 | 稳定范式 | #10 |
| 汇率跳跃 → 不可对冲风险 → 出口 | 倾向性 | #7 |
| 政策改革 → 政府激励变化 → 税收征管 → 企业行为 | 稳定范式 | #12, #13 |
| 法律制度 → 融资约束缓解 → 企业升级 | 倾向性 | #14 |
| 激励/治理 → 管理者行为 → 产品市场 → 利润 | 倾向性 | #9 |

### 常见异质性维度

| 异质性变量 | 稳定性 | 解释范式 |
|---|---|---|
| 企业所有制（国有/民营） | 稳定范式 | 制度逻辑（激励差异、预算软约束） |
| 融资约束 | 稳定范式 | 金融摩擦放大/缩小冲击效应 |
| 企业规模/行业特征 | 倾向性 | 资源禀赋差异 |
| 地区制度环境（市场化、司法） | 倾向性 | 制度互补/替代 |

## 2. 数据

### 数据层级与来源

| 数据层级 | 典型来源 | 稳定性 |
|---|---|---|
| 企业-年份 | CSMAR / 中国工业企业数据库 / Wind | 稳定范式 |
| 行业-年份 | 中国统计年鉴 / 海关贸易数据 | 倾向性 |
| 国家-年份（跨国/双边） | IMF IFS / WDI / Penn World Table | 倾向性 |
| 交易级（产品-企业-年份） | 中国海关数据库 | 倾向性（#14） |

### 数据使用特点
- 偏好**大样本面板数据**（中国工业企业数据库出现频率最高）
- 时间区间通常跨越 8–15 年
- 样本筛选规范：剔除金融企业/ST/缺失值，1%缩尾处理
- 区分"固定权重"（避免内生性）和"时变权重"

## 3. 识别策略

### 常用识别策略

| 策略 | 使用场景 | 稳定性 | 论文 |
|---|---|---|---|
| DID / 多时点DID | 政策冲击（法律、改革） | 稳定范式 | #11, #12, #13, #14, #15 |
| 面板固定效应 + 分组 | 企业异质性分析 | 稳定范式 | #6, #8, #9, #10 |
| GMM (System/Diff) | 动态面板（投资、资本结构） | 倾向性 | #9, #10 |
| GARCH/ARJI | 汇率/金融资产波动 | 倾向性 | #6, #7, #8 |
| PPML 重力模型 | 贸易方程（有零值） | 单篇但规范 | #7 |
| VAR/协整 | 时间序列宏观 | 早期 | #2, #3 |

### 识别改进趋势

早期（2000–2005）：OLS/协整 → 描述性 → 弱因果
中期（2011–2015）：面板FE + 分组 → 相关性证据
近期（2016–2024）：DID/自然实验 → 因果推断 → **强因果**

### 内生性处理标配

| 处理手段 | 稳定性 |
|---|---|
| 行业/企业层面变量分离（缓解反向因果） | 稳定范式 |
| 滞后期自变量 | 稳定范式 |
| 固定效应（企业/行业/年份） | 稳定范式 |
| 自然实验 + DID | 稳定范式（近期） |
| 安慰剂检验、平行趋势 | 稳定范式（近期 DID） |
| PSM-DID 稳健性 | 倾向性 |

### 因果表述边界

李广众论文的因果表述遵循严格的层级：

| 方法 | 典型表述 |
|---|---|
| DID/自然实验 | "X causally affects Y" / "X导致Y"→ 最强 |
| 面板 FE + 分组 | "evidence suggests" / "发现X与Y显著相关"→ 中等 |
| 时间序列 OLS | "X与Y存在相关性"→ 最弱 |

## 4. 稳健性检验体系

| 检验类型 | 稳定性 | 举例 |
|---|---|---|
| 替换 Y 度量 | 稳定范式 | 换指标口径 |
| 替换 X 度量 | 倾向性 | 换代理变量 |
| 子样本排除 | 稳定范式 | 排除特定行业/地区/年份 |
| 替换模型设定 | 倾向性 | OLS→FE→GMM |
| 安慰剂检验 | 稳定范式（DID系列） | 虚构处理组/虚构政策时间 |
| PSM 匹配后回归 | 倾向性 | — |
| 动态效应/事件研究 | 稳定范式（DID系列） | 逐年系数检验 |
