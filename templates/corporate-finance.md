# 公司金融领域专用蒸馏模板

本模板用于蒸馏公司金融领域的学者研究范式。

## ⚠️ 触发条件（必须同时满足）

本模板**仅在以下条件同时满足时**才可使用：

1. **领域条件**：论文属于公司金融领域（企业投资/融资/治理/并购/盈余管理）
2. **作者条件**：论文来自同一作者（或用户明确要求蒸馏某位学者）

如果论文来自不同作者，或者用户没有明确要求蒸馏，**不得使用本模板**，应使用对话确认方式选择其他精炼方式。

---

## 1. 领域特征

### 1.1 核心研究对象

公司金融研究的核心对象：
- **企业**（上市公司、非上市公司、国有企业、民营企业）
- **投资者**（机构投资者、个人投资者、外国投资者）
- **债权人**（银行、债券持有人）
- **监管机构**（证监会、交易所）

### 1.2 典型 X（核心解释变量）

| X 类型 | 示例 | 稳定性 |
|---|---|---|
| 公司治理冲击 | 独立董事政策、股权激励、反收购条款 | 稳定范式 |
| 融资约束变化 | 信贷政策、利率市场化、金融发展 | 稳定范式 |
| 监管政策 | 会计准则变更、信息披露要求、退市新规 | 稳定范式 |
| 市场冲击 | 股价崩盘风险、市场情绪、分析师关注 | 倾向性 |
| 制度环境 | 产权保护、司法质量、政府干预 | 倾向性 |

### 1.3 典型 Y（被解释变量）

| Y 类型 | 示例 | 稳定性 |
|---|---|---|
| 企业投资 | 固定资产投资、R&D 投资、并购 | 稳定范式 |
| 企业融资 | 融资成本、杠杆率、现金持有 | 稳定范式 |
| 企业行为 | 盈余管理、避税、信息披露质量 | 稳定范式 |
| 企业绩效 | 托宾 Q、ROA、ROE、生产率 | 稳定范式 |
| 企业价值 | 市值、长期回报、风险 | 倾向性 |

### 1.4 典型 M（机制渠道）

| 机制链条 | 稳定性 |
|---|---|
| 治理改善 → 代理成本降低 → 企业价值提升 | 稳定范式 |
| 融资约束缓解 → 投资增加 → 企业成长 | 稳定范式 |
| 信息披露增强 → 监督加强 → 代理成本降低 | 稳定范式 |
| 风险管理 → 融资成本降低 → 投资增加 | 倾向性 |

### 1.5 典型异质性维度

| 维度 | 理论依据 | 稳定性 |
|---|---|---|
| 企业所有制（国有/民营） | 代理问题差异 | 稳定范式 |
| 融资约束程度 | 投资对现金流敏感性 | 稳定范式 |
| 治理结构 | 代理成本差异 | 稳定范式 |
| 行业特征 | 竞争程度、资产专用性 | 倾向性 |
| 地区制度环境 | 产权保护、金融发展 | 倾向性 |

## 2. 数据诊断协议

### 2.1 常用数据来源

| 数据层级 | 典型来源 | 稳定性 |
|---|---|---|
| 企业-年份 | CSMAR、Wind、工业企业数据库 | 稳定范式 |
| 企业-季度 | CSMAR 季报数据 | 稳定范式 |
| 企业-事件 | 并购数据、IPO 数据 | 倾向性 |
| 行业-年份 | 行业统计年鉴 | 倾向性 |

### 2.2 样本构建检查

| 项目 | 检查问题 |
|---|---|
| 样本单位 | 上市公司？工业企业？非上市企业？ |
| 时间区间 | 是否至少 8-10 年？ |
| 排除规则 | ST？金融企业？缺失值？ |
| 主键 | 合并主键是否清楚？ |
| 观测数 | 最终样本量是否足够？ |

### 2.3 变量构造检查

| 变量类型 | 常见问题 |
|---|---|
| Y（企业绩效） | 托宾 Q 的计算口径是否一致？ |
| X（核心解释变量） | 是否有测量误差？ |
| 控制变量 | 是否包含 bad controls？ |
| 机制变量 | 是否只是另一个结果变量？ |

## 3. 识别诊断协议

### 3.1 常见内生性来源

| 来源 | 检查问题 |
|---|---|
| 遗漏变量 | 是否控制了企业固定效应？ |
| 反向因果 | X 是否受 Y 影响？ |
| 测量误差 | 核心变量如何构造？ |
| 样本选择 | 样本筛选是否合理？ |

### 3.2 识别策略选择

| 场景 | 推荐策略 | 公司金融案例 |
|---|---|---|
| 政策冲击（清晰处理/对照组） | DID / 多时点DID + 事件研究 | 独立董事政策、会计准则变更 |
| 企业面板（无清晰冲击） | 面板 FE + 分组异质性 | 治理结构与企业绩效 |
| 动态面板（被解释变量有惯性） | System GMM | 杠杆率、现金持有 |
| 企业事件（并购、IPO） | 事件研究 + DID | 并购绩效、IPO 折价 |

### 3.3 因果表述规则

| 识别强度 | 允许表述 | 禁止表述 |
|---|---|---|
| 强（DID/自然实验） | X causally affects Y | X 完全决定了 Y |
| 中（面板FE+分组） | evidence suggests | X 因果导致 Y |
| 弱（OLS） | X is associated with Y | 因果影响 |

## 4. 稳健性检验标配

### 4.1 公司金融稳健性检验清单

| 检验类型 | 做法 | 稳定性 |
|---|---|---|
| 替换 Y 度量 | 换用不同的企业绩效/投资/融资度量 | 稳定范式 |
| 替换样本 | 剔除特殊样本（如金融危机年份） | 稳定范式 |
| 安慰剂检验 | 随机分配处理组/时间 | 倾向性 |
| 工具变量 | 使用外生政策冲击作为 IV | 倾向性 |
| Heckman 两步 | 处理样本选择问题 | 倾向性 |

### 4.2 机制检验清单

| 机制类型 | 检验方法 | 稳定性 |
|---|---|---|
| 代理成本 | 分组回归（高/低治理） | 稳定范式 |
| 融资约束 | 分组回归（高/低融资约束） | 稳定范式 |
| 信息不对称 | 分组回归（高/低信息披露） | 倾向性 |

## 5. 通用实证论文工作流

### 5.1 选题诊断

检查清单：
1. 是否有清楚的经济主体（企业/投资者/债权人）？
2. 是否有明确的 X、Y 和可解释的 M？
3. 是否有具体的公司金融场景（而非宏大口号）？
4. X 是否有可信的外生变异来源？
5. 数据是否可得（CSMAR/Wind/工业企业）？
6. 是否能用 DID/面板 FE 等方法做因果推断？

### 5.2 文献综述

组织方式：
- 按理论分支组织（代理理论/融资约束/公司治理）
- 每支概括后指出与本文关系
- 精准指出"前人未做什么"

### 5.3 数据与变量

推荐数据源：
- CSMAR（上市公司财务数据）
- Wind（金融市场数据）
- 工业企业数据库（非上市企业）

### 5.4 模型与识别

推荐策略：
- 面板 FE + 聚类标准误
- DID + 事件研究
- System GMM（动态面板）

### 5.5 回归表解释

先报告经济显著性（系数含义），再讨论统计显著性。

### 5.6 稳健性检验

- 替换 Y 度量
- 替换样本
- 安慰剂检验

### 5.7 机制检验

检查机制链条和 proxy 质量。

### 5.8 异质性检验

按理论预测分组，检验组间差异。

### 5.9 论文写作

Introduction / 文献 / 结果段落。

### 5.10 审稿式修改

先结构后语言。

---

## 6. 禁止写法

1. 空泛政策套话
2. 没有机制和证据的宏大判断
3. 不编造文献引用
4. Introduction 写成背景材料堆砌
5. 把相关性说成因果
6. 为了显著性建议删样本、换口径或 p-hacking
7. 把机制检验写成因果证明
8. 把异质性检验做成随意分组
